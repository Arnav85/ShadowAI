Production-Grade AI Safety Gateway| Full-stack Python 3-Tier Architecture | Automated Data Privacy and Audit Login


ShadowAI: Enterprise AI Governance Gateway
The Problem: Companies want to use LLMs but are afraid of leaking sensitive data (PII) like customer emails, credit card numbers, or database credentials.
The Solution: An automated 3-tier gateway that "intercepts" prompts, masks sensitive info, and logs every transaction for compliance.
🏗️ Architecture
Frontend: Streamlit dashboard for interactive prompt testing and real-time masking previews.
Backend: FastAPI engine with custom NLP logic for PII detection and secure API routing.
Database: PostgreSQL (with SQLite for local dev) for high-speed audit logging and session management.
Infrastructure: Containerized with Docker and ready for AWS/Azure deployment via Terraform.
